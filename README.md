# PresentationCard
iOS app that shows my contact information.

Concepts:

* SwiftUI
* Arrange elements to create complex layouts
* Add custom fonts
* RGB Colors and Hex codes
* SF Symbols
* Extract subviews to create reusable SwiftUI components
